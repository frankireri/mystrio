# Features for Mystrio App

## Core App Features:

*   **User Authentication & Profile:**
    *   Users can create a unique username.
    *   Each profile has a unique, shareable link (e.g., `mystrio.app/your-username`).
    *   **Theme Toggle:** Users can switch between light and dark mode, with preference saved.
*   **Anonymous Questions (Inbox):**
    *   Anyone with the user's profile link can send them an anonymous question.
    *   Users view received questions in a dedicated inbox.
    *   Users can publicly answer questions.
    *   **Sharing Answers:** Users can share their answered questions as images.
*   **Customizable Question Cards:**
    *   Users can customize their question cards with different templates and designs.
    *   **Animated Backgrounds:** Subtle animations on card backgrounds.
    *   **Font Styles:** Different font families applied based on selected card style.
    *   **Image Backgrounds:** Option to upload custom images for card backgrounds.
    *   **Interactive "Dice Roll":** A button to randomly generate a new question and style combination.
*   **Sharing Profile Link:**
    *   Users can share their profile link on social media to get more questions, with a visually integrated link on the shared image.

## "How Much Do You Know Me?" Quiz Game:

*   **Quiz Creation & Management:**
    *   Users can create and name multiple custom quizzes.
    *   Add, edit, and remove quiz questions (question, correct answer, dummy answers).
    *   **Suggested Questions:** Predefined questions available based on selected theme.
    *   **Themed Quizzes:** Choose a theme for the quiz, applying a unique design (gradient colors) to the creation, play, and leaderboard pages.
*   **Quiz Playing:**
    *   Friends can take the quiz.
    *   **Scoring:** Tracks the player's score.
    *   **Engaging Score Reveal:** Confetti animations celebrate the score.
    *   **Score-Based Avatars:** Different avatars/messages displayed based on the final score.
*   **Leaderboard:**
    *   A leaderboard to show who scored the highest for each quiz.
*   **Quiz Sharing:**
    *   Share the quiz with friends via a unique link.

## "Gratitude Jar" Feature:

*   **Gratitude List Creation:**
    *   Users can add and manage a list of things they are grateful for.
    *   **Custom Header:** Users can define a custom header for their list (e.g., "Things I'm Grateful For," "My New Year's Resolutions"), with predefined options for quick selection.
*   **Customizable Gratitude Cards:**
    *   **Themed Cards:** Select from various themes (e.g., "Serene," "Joyful," "Cozy") with unique gradients, icons, and colors.
    *   **Custom Image Backgrounds:** Upload personal images for the card background.
    *   **Interactive Header:** Drag and resize the custom header text on the card, with position and scale saved.
    *   **Readability Overlay:** Semi-transparent overlay ensures text is readable on image backgrounds.
*   **Sharing Gratitude:**
    *   Share the gratitude list as a beautifully designed image.

## User Flow Improvements:

*   **Clear Onboarding:** New users are guided from username creation to a `GameSelectionPage` to choose their first activity.
*   **Intuitive Navigation for Returning Users:**
    *   Dedicated `MainTabPage` with a bottom navigation bar.
    *   **Home Dashboard:** A central hub showing an overview of ongoing activities (quizzes, gratitude).
    *   **Inbox:** Dedicated tab for anonymous questions.
    *   **Create:** Dedicated tab to access all content creation tools (Q&A, Quiz, Gratitude).
    *   **Profile/Settings:** Dedicated tab for account management and app settings.

## Future Android App Features (from original plan):

*   Push Notifications: To notify users of new questions.
*   Deeper Social Media Integration: For easier sharing.
